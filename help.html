Interface

  1. Menu Bar
  2. Operation Panel: Main labeling operations can be accessed here
  3. Labeling Panel: Showing image and allow labeling
  4. Console Output: Where program output shows
In the following part of this tutorial, panel.1-4 will be used to represent these interface panels.

Example - MRILabeling of articular disc

Open DICOM files

To open dicom files for labeling, go to panel.1 -> File -> Open and choose the directory to open.
The directory to be opened can contain multiple patient images and should be structed as follows:
Folder_to_open
|
|--Patient0
|      |--slice0.dcm
|      |--slice1.dcm
|      |--slice2.dcm
|      ∟...
|
|--Patient1
|      |--slice0.dcm
|      ∟...
|
|--Patient2
|      ∟...
∟...

Before start

Before we start labeling, we need to setup labler's name, it's in panel.1 -> Settings -> Labeler name. Notice that it's better for labeler's name to be consistent through out the labeling process, i.e. one person just use one name.
We can also setup the output path of our labeling at panel.1 -> Settings -> Output path, however for this specific project, we do NOT need to change output path, the default output path will be under each patient folder

Navigation

Navigation operation including roaming between patients, slices and different image series of single patient, they are all in the upper part of panel.2. Operations are also binded with mouse buttons:
  • Move image - Right button pressd and move
  • Change slices - Wheel rolling
  • Zoom in/out - Ctrl + Wheel rolling
  • Change contrast/brightness - Middle button pressed and move
  • Initlate labeling - left button pressed and move
Navigation shortcut can be found in panel.1 -> Operations.

Label

Then we start labeling. Labels can be found in panel.1.
By drawing on panel.4 with left mouse button pressed, we started labeling, the curve will be initiated when button is released.
