Example - MRILabeling of articular disc

Open DICOM files

To open dicom files for labeling, go to panel.1 -> File -> Open and choose the directory to open
the directory to be opened can contain multiple patient images and should be structed as follows:
Folder_to_open
|--Patient0
|      |--slice0.dcm
|      |--slice1.dcm
|      |--slice2.dcm
|      |--...
|--Patient1
|      |--slice0.dcm
|      |--...
|--Patient2
|      |--...
|--...

Before start

Before we start labeling, we need to setup labler's name, it's in panel.1 -> Settings -> Labeler name.
Notice that it's better for labeler's name to be consistent through out the labeling process, i.e. one person just use one name.
